# Tkram Test Project
a mini project that was meant to test my comprehensive knowledge in back development in general and laravel framework in particular


## Local Development Setup:  
1. Clone the repository:
   ```
   git clone https://github.com/amro-baira23/source-safe.git
   ```
2. Start local server:
   ```
   php artisan serve 
   ```
*. Run Tests:
   ```
   php artisan test 
   ```
